# Scene Rules

After each written scene, provide five recommended next scenes in the most appropriate chronological or dramatic order:

The set should normally contain:

1. One plot-advancing beat
2. One escalation beat
3. One valid complication beat
4. Two useful side-beats

`Proceed`, `Continue`, `Repeat`, and equivalent commands implement all five current recommendations as complete scenes, braided where useful, before producing exactly five new recommendations.

Complications must respect established capabilities. Do not use casual detection of Ruuk's passive mobility augment or NeuralComm.

Use **Roamer** in prose, **Garnette** as the ship name, and avoid new names beginning with V. Keep travel time, jump week, fuel, law, sensors, money, injury, and logistics visible.
