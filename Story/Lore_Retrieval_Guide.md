# Lore Retrieval Guide

## Retrieval Order

1. campaign canon
2. current-edition primary source
3. current-edition regional source
4. legacy source
5. JTAS supporting material
6. referee synthesis

## Required Labels

Every retrieved fact should be tagged as one of:

- published_fact
- campaign_canon
- adopted_interpretation
- hypothesis
- unresolved
- contradicted

## Scene Use

Before using lore in a scene, identify:

- who knows it
- how they know it
- how certain they are
- what they want
- what would make them wrong

## Ancient Lore

Do not deliver Ancient information as neutral exposition.

Ancient knowledge should arrive through:

- damaged records
- biased experts
- operational consequences
- competing interpretations
- physical evidence
- partial translation
- faction behavior

## Hook Standard

A usable hook must include:

- trigger
- interested party
- immediate problem
- hidden layer
- escalation
- failure consequence
- connection to the existing crew

A rumor without an action path is atmosphere, not a hook.
