# Traveller Campaign Brain

A living, Git-versioned knowledge base for the **Mysterious Secrets of Wrathful Ancients** Traveller campaign.

## Purpose

This repository separates published-source facts, campaign canon, interpretation, and live state. Transcripts are evidence. Campaign files are curated conclusions.

## Canon Priority

1. Explicit referee ruling
2. Actual-play transcript
3. Current campaign-state files
4. Published adventure material
5. Earlier generated prose
6. Speculation

When sources conflict, preserve the conflict in `Campaign/Continuity_Issues.md` until resolved.

## Core Cast

Ruuk and Dralan are co-protagonists. Tapir Tracks, Solveig Dahl, Ardric Vellarn D'Ravayne, Corbin Vanis, Roamer, and Mercedes Taramaka are continuing ensemble protagonists whose arcs evolve through play.

## Workflow

After each session, update chronology, live state, character evolution, relationships, mysteries, and continuity issues, then generate five chronologically sensible next-scene recommendations.

## Copyright

Do not copy published Traveller text into this repository. Store concise derived notes, page references, and campaign-facing interpretations.
