# Project Operating Manual

## Meaning of command words

### Proceed / Continue / Repeat

When the user says `Proceed`, `Continue`, `Repeat`, `Repeat last instruction`, or equivalent:

- use the most current set of five recommendations
- write all five recommendations as complete prose scenes
- braid scenes together when that improves continuity or dramatic structure
- organize scenes chronologically unless another order is clearly stronger
- preserve state changes from each completed scene
- do not ask the user to choose among recommendations
- after completion, provide exactly five new recommended next scenes

## Five-beat recommendation mixture

The five recommended scenes should normally contain:

- one plot-advancing beat
- one escalation beat
- one complication beat
- two useful side-beats

All prose and recommendations remain subject to the active campaign state, branch policy, character knowledge, and routed canon sources.
