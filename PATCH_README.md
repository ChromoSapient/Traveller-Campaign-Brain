# Traveller Campaign Brain v0.4: Knowledge Foundation

This patch establishes the edition-aware Traveller Knowledge Graph.

## Included

- Canon methodology
- Knowledge-fact schema with provenance
- Edition index
- Source inventory structure
- Lore convergence workflow
- Adoption-status rules
- Dependency tracking
- Initial Vargr, robotics, Darrian, and Imperial nobility knowledge files
- JTAS indexing schema and intake queue
- v0.4 roadmap and extraction log

## Install

Copy the folders in this archive into the repository root.

Suggested commit message:

`Add v0.4 Traveller knowledge foundation`
